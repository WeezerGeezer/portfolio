    Arduino Catchphrase System

    Complete Hardware Game with iOS Companion

    Built with Arduino C++, Swift, Bluetooth LE, and Hardware Design

    Hardware Code Watch Demo

    Project Overview

    A complete hardware game system that reimagines the classic Catchphrase party game with modern technology. Features extended battery life, wireless content management through a custom iOS app, and professional hardware design for an enhanced gaming experience that addresses the limitations of commercial alternatives.

    The Challenge

    Building a superior party game system required solving multiple hardware and software integration challenges:

    • Power Management: Achieving 60+ hour battery life while maintaining responsive performance and visual feedback
    • Wireless Communication: Creating reliable Bluetooth LE connection between iOS app and Arduino device
    • Content Management: Designing intuitive wireless system for adding and organizing game categories and phrases
    • Hardware Integration: Combining microcontroller, display, storage, and user interface in compact, durable design

    Technical Solution

    Advanced Power Management

    Implemented deep sleep modes and optimized power circuits achieving 60-80 hour battery life, with intelligent wake/sleep cycles preserving energy during inactive periods.

    Bluetooth LE Integration

    Built custom BLE service architecture enabling seamless communication between iOS app and Arduino device with error handling and reconnection protocols.

    Modular Hardware Design

    Designed scalable PCB layout with clear pin allocation, replaceable components, and expansion capabilities for future feature additions.

    Cross-Platform Development

    Developed synchronized ecosystem spanning C++ firmware, Swift iOS application, and JSON data protocols for unified user experience.

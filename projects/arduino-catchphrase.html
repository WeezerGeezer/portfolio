    Arduino "Catchphrase" Device

    Custom Hardware Game Recreation

    Built with C++ & Swift App

    Watch Demo View Code

    Project Overview

    A physical recreation of the popular Catchphrase party game using Arduino hardware, featuring custom electronics and a companion iOS app for game management. This project combines embedded systems programming, hardware design, and mobile app development to create an engaging party game experience.

    The Challenge

    Recreating a beloved party game in hardware form presented several unique engineering challenges:

    • Hardware Design: Creating reliable custom electronics that could handle rapid button presses and game timing
    • Real-Time Performance: Ensuring the device could maintain accurate timing and responsive user interactions
    • Power Management: Designing for portable use with efficient battery consumption
    • User Experience: Maintaining the fast-paced, intuitive gameplay of the original while adding new functionality
    • Durability: Building hardware robust enough for enthusiastic party game sessions

    Technical Solution

    Arduino Hardware

    Custom-built Arduino-based device with carefully selected components for optimal performance, including buttons, display, speaker, and LED indicators.

    C++ Firmware

    Low-level C++ programming for Arduino microcontroller, handling game logic, timer management, and real-time user input processing.

    iOS Companion App

    Swift-based iOS application for game configuration, word database management, and device communication via Bluetooth.

    Game Logic Engine

    Sophisticated state machine managing game flow, randomization, scoring, and timer functionality for seamless gameplay.

    Hardware Components

    Arduino Microcontroller

    Arduino Uno as the main processing unit, chosen for reliability and extensive documentation support

    LCD Display

    16x2 character LCD for displaying words, timer, and game status information

    Input Controls

    Tactile push buttons for game control, start/stop functionality, and team switching

    Audio Output

    Piezo buzzer for timer alerts, sound effects, and audio feedback during gameplay

    Visual Indicators

    RGB LEDs for game state indication, team colors, and timer countdown visualization

    Power Management

    Battery pack with power regulation for portable operation and consistent performance
